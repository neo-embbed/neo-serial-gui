# 监测卡片使用指南

## 概述

监测卡片用于从串口通信数据中**实时提取并展示关键参数**。每张卡片绑定一条正则匹配规则，当接收到的数据命中该规则时，卡片上的数值会自动更新。

---

## 创建卡片

点击卡片区域右上角的 **+** 按钮打开「新建监测卡片」弹窗。

弹窗支持两种模式，可通过右上角按钮切换：

---

### 简单模式（默认）

适合不熟悉正则表达式的用户。只需填写串口数据中数值/关键词的**前缀**和**后缀**，程序会自动生成匹配规则。

#### 字段说明

| 字段 | 说明 |
|------|------|
| **名称** | 卡片显示标题，如「温度」「电压」 |
| **前缀** | 数值前面的固定文本，如 `temp=` |
| **类型** | 下拉选择「数值」或「布尔」 |
| **后缀** | 数值后面的固定文本，如 `°C` |
| **支持负数** | 仅数值类型可见，勾选后可匹配 `-25.3` 这样的负数 |
| **单位** | 卡片上显示的单位标签 |
| **颜色** | 卡片主题色，十六进制格式如 `#ff6600` |

#### 数值型示例

假设串口会收到这样的数据：

```
temp=25.3°C
```

填写方式：
- 前缀：`temp=`
- 类型：数值
- 后缀：`°C`
- 支持负数：如果可能收到 `temp=-5.0°C`，则勾选

程序自动生成的正则：`temp=(-?\d+\.?\d*)°C`

弹窗底部会显示「生成: ...」预览最终的正则表达式。

#### 布尔型示例

假设串口会收到这样的数据：

```
pump=ON
pump=OFF
```

填写方式：
- 前缀：`pump=`
- 类型：布尔
- 后缀：（留空）

选择布尔类型后，下方会出现 **True 关键词**和 **False 关键词**列表：

- True 关键词：填入代表「真」的文本，如 `ON`、`YES`、`1`
- False 关键词：填入代表「假」的文本，如 `OFF`、`NO`、`0`

每个列表都可以通过 **+** 按钮追加更多关键词，用 **x** 按钮删除。

程序自动生成的正则：`pump=(ON|OFF); true=ON; false=OFF`

---

### 高级模式

适合熟悉正则表达式的用户，可以直接手写匹配规则。

#### 字段说明

| 字段 | 说明 |
|------|------|
| **名称** | 卡片显示标题 |
| **匹配模式** | 完整的正则表达式 |
| **类型** | 选择「数值」或「布尔」 |
| **单位 / 颜色** | 同简单模式 |

#### 正则书写规则

**数值型：** 使用捕获组 `()` 括住要提取的数字部分。如果没有捕获组，则使用整个匹配结果。

```
temp=(\-?\d+\.?\d*)           匹配 temp=25.3 → 提取 25.3
Volt=([0-9.]+)V               匹配 Volt=3.3V → 提取 3.3
```

**布尔型：** 正则后面用分号分隔附加 true/false 映射：

```
<正则>; true=关键词1|关键词2; false=关键词3|关键词4
```

示例：
```
status=(\w+); true=ON|YES|1; false=OFF|NO|0
```

如果省略 `true=` / `false=`，将使用默认映射：
- True：`ON`、`TRUE`、`1`、`YES`
- False：`OFF`、`FALSE`、`0`、`NO`

## 卡片操作

### 移动
拖拽卡片**顶部标题栏**即可移动位置。

### 缩放
拖拽卡片边缘可调整大小。卡片内文字会随缩放自动调整字号。最小尺寸为 140x100。

### 置顶
点击任意卡片会自动将其置于最顶层（不需要松开鼠标就会生效）。

### 删除
点击卡片标题栏右上角的 **x** 按钮删除卡片。

---

## 数据持久化

- 卡片配置自动保存到 `./data/monitor_cards.json`
- 创建或删除卡片时会立即写入文件
- 下次启动程序时自动加载

---

## 常见问题

**Q：卡片一直显示「等待数据...」？**
- 确认串口已连接并有数据接收
- 检查正则表达式是否正确（可在弹窗底部查看「生成:」预览）
- 在通信日志中确认收到的数据格式与预期一致

**Q：布尔卡片显示「不匹配」？**
- 正则捕获到了文本，但该文本不在 True/False 关键词列表中
- 检查关键词拼写是否与实际数据一致（匹配时不区分大小写）

**Q：如何匹配更复杂的数据格式？**
- 切换到高级模式，手写正则表达式
- 示例：`T21: ADC=0x[0-9A-Fa-f]+, Volt=[0-9.]+V, Temp=([-+]?\d+\.\d+)°C`
